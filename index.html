Conhecendo melhor o Git

O Git é um sistema de controle de versão distribuído, criado por Linus Torvalds em 2005, que se tornou uma ferramenta fundamental no desenvolvimento de software. Sua principal função é permitir que vários desenvolvedores trabalhem no mesmo projeto de forma colaborativa, mantendo um histórico detalhado de todas as alterações feitas no código-fonte.

Comandos:

  • Commit
  • Push
  • Pull
  • Sleep
  • End
